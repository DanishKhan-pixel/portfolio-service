# Frontend - React + Vite

## Setup

```bash
cd frontend
npm install
cp .env.example .env
```

## Run

```bash
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Environment

| Variable | Default |
|----------|---------|
| `VITE_API_BASE_URL` | `http://localhost:8000/api` |

Frontend expects backend running at `http://localhost:8000`.
